# Battaglia-Navale

Project fully developed in C++, implementing the classic Battleship game. The program allows the user to challenge the CPU, which randomly places ships of different lengths on a 10x10 grid. The goal is to guess the ship positions using the fewest possible attempts. The game tracks hits and misses in real time, updates the remaining ships counter, and prevents repeated coordinate entries, providing a complete console-based gameplay experience.




Progetto sviluppato interamente in C++, che implementa il classico gioco della Battaglia Navale.
